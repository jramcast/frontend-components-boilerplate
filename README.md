# frontend-components-boilerplate
A boilerplate for proyects based on components, using webpack, es6, sass and pug.

WORK IN PROGRESS


## Requirements

* Node version >= 6

## Usage

Install dependencies:
```
npm install
```

Run app:
```
npm start
```

Run tests:
```
npm test
```

Run continuous testing:
```
npm run test:watch
```

Run linter (coding standard):
```
npm run lint
```
